Полулюкс с балконом

3 х местный полулюкс с балконом подойдёт для комфортного проживания 2-3 человек, располагает двуспальной кроватью и одной односпальной или кресло кроватью, так же есть варианты с тремя односпальными кроватями. В номере есть небольшой балкон оборудованный столиком и стульями с видом на горы Номер оснащен всем необходимым для комфортного проживания ( кондиционер, телевизор с плоским экраном, холодильник, электрический чайник,шкаф,две тумбы, чайный набор,бокалы для вина. В собственной ванной комнате есть фен, набор белоснежных полотенец, халат, тапочки, шампунь-гель для душа, жидкое мыло, косметический набор, шапочка для душа, туалетная бумага, гигиенический набор. В номере возможна установка дополнительного спального места в виде евро раскладушки (бесплатно)
